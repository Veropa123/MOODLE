# Historia Usuario 3 - Inventario Python

Este proyecto gestiona un inventario de productos desde consola.
Funcionalidades:
- CRUD completo (Agregar, Mostrar, Buscar, Actualizar, Eliminar)
- Estadísticas del inventario
- Guardar y cargar CSV
- Menú interactivo con validación de entradas

Estructura de carpetas:
- app.py
- servicios.py
- archivos.py
- README.md
- diagrama_flujo.png
